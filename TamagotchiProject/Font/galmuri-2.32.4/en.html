Galmuri

Hangul bitmap fonts
with good readability and balanced
even in small size

Scroll down to see more

Bitmap fonts based on the font design from Nintendo DS. With additional characters support.

All Hangul syllables except for 2,355 Hangul syllables in Galmuri11 and Galmuri9, all Hangul glyphs in Galmuri7 and All glyphs in Galmuri11 Bold and Galmuri11 Condensed are my pure creations.

Galmuri only look best at 12px size for Galmuri11, 10px size for Galmuri9, 8px size for Galmuri7, or at integer multiples thereof.

Use as Web Fonts

HTML

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/galmuri@latest/dist/galmuri.css">

CSS

@import url('https://cdn.jsdelivr.net/npm/galmuri@latest/dist/galmuri.css');

CSS Rules

/* To use Galmuri11 */
font-family: 'Galmuri11';

/* To use Galmuri11 Bold */
font-family: 'Galmuri11';
font-weight: 700;

/* To use Galmuri11 Condensed */
font-family: 'Galmuri11';
font-stretch: condensed;

/* To use Galmuri9 */
font-family: 'Galmuri9';

/* To use Galmuri7 */
font-family: 'Galmuri7';
