Galmuri

작은 크기에서도
가독성 좋고 균형 있는
한글 비트맵 폰트

아래로 스크롤하여 더 많은 내용 확인하기

Galmuri는 닌텐도 DS 본체와 소프트웨어에 사용되었던 폰트 디자인을 본따 만들어진 비트맵 폰트입니다. 2019년 10월 9일 한글날에 처음 공개되었으며, Galmuri의 이름은 2008년 6월 사용자 한글화 커뮤니티인 한식구에서 김동한 님께서 만들어 배포하신 비트맵 폰트 ‘갈무리M’에서 유래하였습니다.

현재 Galmuri11, Galmuri9는 모든 한글 음절 11,172자와 JIS 한자 6,355자를 지원하며 Galmuri11 Bold는 Adobe-KR-0의 한글 음절 2,780자, Galmuri11 Condensed는 모든 한글 음절 11,172자, Galmuri7은 Adobe-KR-0과 Adobe-KR-1의 한글 음절 4,358자와 JIS 한자 6,355자를 지원합니다.

Galmuri11과 Galmuri9의 경우 한글 음절 중 2,355자를 제외한 모든 글리프, Galmuri7의 경우 모든 한글 글리프, Galmuri11 Bold와 Galmuri11 Condensed의 경우 모든 글리프가 Nintendo와 관련이 없는 저의 순수한 창작물입니다.

라이선스

Copyright © 2019-2022 Minseo Lee (itoupluk427@gmail.com), with reserved font name “Galmuri”.

Galmuri는 SIL 오픈 폰트 라이선스 1.1에 따라 사용할 수 있으며, 폰트가 자체적으로 판매되지 않는 한 자유롭게 사용·연구·수정·재배포할 수 있습니다. 또한 어떠한 경우에도 저작권자는 계약·불법 행위 또는 기타 계약의 조치로 인한 일반·특수·간접·부수·결과적 손해를 포함하여 어떠한 청구·손해 또는 기타 책임도 지지 않습니다.

그리고 Galmuri를 수정하여 파생된 이차적 저작물 폰트는 이름에 ‘Galmuri’를 사용할 수 없으며, 다른 유형의 라이선스로 배포할 수 없습니다.

OFL 1.1을 한국어로 번역한 내용은 이곳에서 확인할 수 있으며, 라이선스 원문은 이곳에서 확인할 수 있습니다.

다운로드

GitHub 릴리스 페이지에서 다운로드하기

Galmuri11은 12px, Galmuri9는 10px, Galmuri7은 8px 크기와 그 배수에서 가장 명확하게 표시됩니다.

웹폰트로 사용

HTML에서:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/galmuri@latest/dist/galmuri.css">

CSS에서:

@import url('https://cdn.jsdelivr.net/npm/galmuri@latest/dist/galmuri.css');

CSS 규칙:

/* Galmuri11을 사용하려면 */
font-family: 'Galmuri11';

/* Galmuri11 Bold를 사용하려면 */
font-family: 'Galmuri11';
font-weight: 700;

/* Galmuri11 Condensed를 사용하려면 */
font-family: 'Galmuri11';
font-stretch: condensed;

/* Galmuri9을 사용하려면 */
font-family: 'Galmuri9';

/* Galmuri7을 사용하려면 */
font-family: 'Galmuri7';
